BPS Workspace Default User Permission Configuration Script

BPS Workspace is a biz process centralization of management workarea that offered by the BPS product of Primeton. You could manage various biz process and optimize job by using the model of organization that supported by BPS product. (Of course, you can replace with BPS opening interface in actual project.)

BPS supported six default internal permission set:

  • bpsclient：Client use permission. It offered the function of start process, view work item, recieve/execute work item, agent setting and so on;
  • bpsadmin：Managed engine instance manage permission. It offered the function of engine instances' register,management,configuration and so on. (This permission is only default possessed of the user sysadmin );
  • bpsmanager：Process manage&monitor permission. You can use operating platform of workflow. Business process management staff not only can manage the process instances, but also can monitor the process action of business operation staff;
  • bpscustom：Biz process configuration permission. You can use the tools of process business composer implement by BPS process engine. Business process management staff could modeling,configuration and adjustment of process in the view of business by using it;
  • bpsconfig：Managed engine instance manage permission. You can use the tools of system management by which system administrator could config and manage the workflow engine;
  • bpsbizcat：Biz catalog management permission. It offered the function of biz catalog's creation,maintain,authorization and the function of import&export of biz process and biz rosource;

BPS supported default user and permission allocation:

tiger： Client use permission, Process manage&monitor permission, Biz process configuration permission, Managed engine instance manage permission, Biz catalog management permission;
goose： Client use permission, Process manage&monitor permission;
fish： Client use permission, Process manage&monitor permission, Biz process configuration permission, Managed engine configuration manage permission;
snoppy： Client use permission, Managed engine instance manage permission, Biz catalog management permission;
micky： Client use permission, Biz catalog management permission;
kitty： Client use permission;
sysadmin： Managed engine instance manage permission;
